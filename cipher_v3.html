How to play

Objective

1

A hidden colour sequence is generated. Fill each slot from the palette, then press Submit.

2

Feedback pegs reveal information. Use deduction to crack the code within the guess limit.

Feedback peg meanings

Purple — correct colour, correct position

Orange — correct colour, wrong position

Grey — colour not in the code

Keyboard shortcuts

1–8 Place colour
Enter Submit
⌫ Remove last
H Use hint
← → Move slot
Del Clear all

Speedrun mode

Unlimited guesses — your only enemy is the clock. Score = 5000 − elapsed seconds. Perfect for pure speed practice.
